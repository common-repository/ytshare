=== YTShare ===
Tags: youtube, share, social, video
Requires at least: 3.0
Tested up to: 3.8
Stable tag: 1.0
License: GPLv2 or later

== Description ==

Embeds a Youtube video onto your blog with the API. Then, when the video is paused or finishes, a popup asks for the user to like the site's page on Facebook.

== Installation ==

Upload the YTShare plugin to your blog and activate it.

Use the shortcode [ytshare videoid="YOUTUBEIDHERE"]
